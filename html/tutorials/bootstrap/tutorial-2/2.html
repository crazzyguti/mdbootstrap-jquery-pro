Our intro looks nice, but heading in the middle barely visible. That can decrease user experience, so we have to do something with that.

This is a perfect occasion to use the mask.

Step 1

Firstly, we will change the color of the text to white. Put the following code into style.css file:


.flex-center {
    color: #fff;
}

Now it looks even worst. We definitely need to make the background darker.

Add a class .hm-black-strong to the .view div.


    <!--Mask-->
    <div class="view hm-black-strong">

Alright, now it's fine.

You can modify the color intensity by changing class .light to .strong or .slight. You can also change the color itself. Try it:


    <!--Mask-->
    <div class="view hm-blue-strong">

All color references you can find in our COLORS DOCUMENTATION. To learn more about masks have a look at our MASKS DOCUMENTATION.

After you finish playing with the colors, change back mask to .hm-black-strong.

Step 2

Let's create a basic structure for our Landing Page.

At the beginning place a container between <main> tags. That will center our content and set up a proper side margins.

We'll also add class .text-center to <main> section to center all the text we'll place in the sections within.


<!--Main layout-->
<main class="text-center">
    <div class="container">
        
    </div>
</main>
<!--Main layout-->

As you can see we've created 3 sections: "About", "Best Features" and "Contact". These will provide a bunch of useful information to visitors of our website.

Each section contains .row to provide a necessary structure for content's .columns which are placed inside. All of them are a part of Bootstrap Grid System. If you are not sure what we are talking aboutÂ you can read more about it in our previous tutorials or our GRID DOCUMENTATION.

In this case, we use .col-md-3 4 times, because we need 4 columns for 4 cards.

Each section has own ID. We'll refer to it later in our navigation.

We've also added title above the sections.
